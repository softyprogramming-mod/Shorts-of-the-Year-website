# Images Folder

Add these images to this folder:

## Required Images

1. **favicon-32x32.png** - 32x32px favicon
2. **favicon-16x16.png** - 16x16px favicon  
3. **Logo.svg** - Your logo for the header

## Tips

- Favicons should be simple, recognizable icons
- Logo should work on dark background (white/light colored)
- Clothing photos should be clear, well-lit
- Recommended size for clothing photos: 1000x1000px minimum

## Creating Favicons

Easy online tools:
- https://favicon.io (convert image to favicons)
- https://realfavicongenerator.net (comprehensive generator)

Just upload your logo and download the generated favicons!
